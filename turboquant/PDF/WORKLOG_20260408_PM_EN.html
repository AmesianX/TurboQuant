Cross-Head WHT Breakthrough Breakthrough

1. Per-Head RMS Norm

  • Before: global L2 norm (shared across 8 heads)
  • Problem: high-energy heads dominated the norm
  • Low-energy heads lost information → garbage
  • Fix: independent RMS norm per head

2. QJL Noise Discovery

  • TBQP3: 2-bit MSE + 1-bit QJL
  • QJL's random projection → K score noise
  • D=256: noise is small (negligible)
  • D=64: noise exceeds signal gap!
Solution: Auto-switch K from tbqp3 → tbq3 at head_dim=64. Invest all 3 bits in Lloyd-Max centroids. SQNR improved 2.3x (3.45 → 7.8).

MMSE Dynamic Softening Theory

D≥256: Sharpening (α > 1)

  • High SQNR → correct token almost always ranks first
  • softmax slightly flattened → corrected by α
  • α(N) = 1 + c × √(ln N)

D=64: Softening (α < 1)

  • Low SQNR → noise flips rankings
  • Sharpening goes all-in on the wrong answer
  • α(N) = SQNR / (SQNR + √(ln N / ln N₀))
