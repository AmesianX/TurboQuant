  • Phase 8: src1 적용 → 35/35 math bench, Multi-turn 성공

Root Cause: src1 Index BUG

set-rows stores at specific KV cache positions via src1[i].
side buffer stored sequentially (0,1,2...) → position mismatch.
ComponentBeforeAfter
KV 캐시src1[0]=5 → position 5Same
Side buffertoken 0 → position 0 ✗token 0 → position 5 ✓
Lesson: Measure, don't guess. Found by directly comparing f16 vs side buffer values.
