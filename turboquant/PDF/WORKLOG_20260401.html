head_dim=64 해결 + Auto Mapping 해결

head_dim=64 해결책

  • K = q8_0 (표준 8-bit)
  • V = tbq3_2 (WHT 3-bit)
  • "볼프강 파울리" 정확 출력 확인
  • 2.7x 압축률 유지

Auto head_dim Mapping

  • 사용자 입력: tbq3 (단축어)
  • GGUF 메타데이터에서 head_dim 자동 감지
  • 256 → _0 / 128 → _1 / 64 → K=q8_0+V=_2
  • 입력 검증 + fallback + 경고 박스
