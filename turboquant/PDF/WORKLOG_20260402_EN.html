V Cache IWHT FP16 Precision Bug Critical

Symptoms

  • ~1500-2300토큰 이후 //// 또는 ??? 반복
  • tbqp3: ~1826tok에서 발생
  • tbqp4: ~2284tok에서 발생
  • f16 V: 문제 없음

Root Cause

  • V IWHT butterfly가 FP16(half)으로 수행
  • Accumulated WHT-domain values → FP16 exceeded precision
  • Catastrophic cancellation in subtraction
  • K-only quant → OK, V-only quant → Corrupted
K/V 분리 테스트가 Core: tbqp3 K + f16 V = OK / f16 K + tbq3 V = Corrupted → V Path 특정

Fix: All WHT/IWHT Converted to FP32 Fixed

PathBeforeAfterImpact
V IWHT (D=256/128/64/576)FP16 butterflyFP32 butterflycorruption Fixed
Q WHT MSE (D=256/128/64/576)FP16 butterflyFP32 butterflyPrecision + ROCm
Q WHT QJL (D=256/576)FP16 butterflyFP32 butterflyPrecision + ROCm
